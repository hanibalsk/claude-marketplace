  Core Workflows

  | Workflow      | Description                                                             |
  |---------------|-------------------------------------------------------------------------|
  | brainstorming | Facilitate interactive brainstorming sessions using creative techniques |
  | party-mode    | Orchestrates group discussions between all installed BMAD agents        |

  Phase 1: Analysis

  | Workflow             | Description                                                         |
  |----------------------|---------------------------------------------------------------------|
  | create-product-brief | Create comprehensive product briefs through collaborative discovery |
  | research             | Conduct comprehensive research (Market, Technical, Domain)          |

  Phase 2: Planning

  | Workflow         | Description                                              |
  |------------------|----------------------------------------------------------|
  | create-ux-design | Work with UX Design expert to plan UI patterns           |
  | create-prd       | Create comprehensive PRD through collaborative discovery |

  Phase 3: Solutioning

  | Workflow                       | Description                                                 |
  |--------------------------------|-------------------------------------------------------------|
  | check-implementation-readiness | Validate PRD, Architecture, and Epics before implementation |
  | create-architecture            | Collaborative architectural decision facilitation           |
  | create-epics-and-stories       | Transform PRD into implementation-ready epics and stories   |

  Phase 4: Implementation

  | Workflow        | Description                                              |
  |-----------------|----------------------------------------------------------|
  | code-review     | Adversarial code review that finds 3-10 issues per story |
  | correct-course  | Navigate significant changes during sprint execution     |
  | create-story    | Create the next user story from epics                    |
  | dev-story       | Execute a story with tasks, tests, and validation        |
  | retrospective   | Review success and lessons after epic completion         |
  | sprint-planning | Generate sprint status tracking file                     |
  | sprint-status   | Summarize sprint status and surface risks                |
